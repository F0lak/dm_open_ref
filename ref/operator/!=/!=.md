## != operator
**See also:**
:   [\> operator](#/operator/%3e)
:   [\< operator](#/operator/%3c)
:   [\<\> operator](#/operator/%3c%3e)
:   [== operator](#/operator/==)
:   [\~= operator](#/operator/~=)
:   [\~! operator](#/operator/~!)
:   [operators](#/operator)
<!-- -->
**Format:**
:   A != B
<!-- -->
**Returns:**
:   1 if A is not equal to B; 0 otherwise
This is identical to the \<\> operator.
To check if A and B are not equivalent, use the \~! operator.
